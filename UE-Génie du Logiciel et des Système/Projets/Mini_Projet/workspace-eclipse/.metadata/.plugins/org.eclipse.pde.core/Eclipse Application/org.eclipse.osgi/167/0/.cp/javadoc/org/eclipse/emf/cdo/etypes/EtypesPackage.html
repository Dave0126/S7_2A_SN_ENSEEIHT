Interface EtypesPackage

  • All Superinterfaces:

    EtypesPackageImpl

    public interface EtypesPackage
extends EPackage
    The Package for the model. It contains accessors for the meta objects to represent
    • each class,
    • each feature of each class,
    • each enum,
    • and each data type
