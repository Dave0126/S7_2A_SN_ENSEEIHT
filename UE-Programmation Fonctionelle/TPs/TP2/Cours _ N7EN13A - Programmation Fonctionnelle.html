    • Généralités

      Tout ce qui se passe en distanciel se déroule sur Discord, serveur des 2A.

      IMPORTANT :
      - Les cours de la première partie (cours 1 à 4) sont en distanciel / asynchrones :
        * Tous les supports sont disponibles sur Moodle
        * L'enseignant sera disponible sur Discord sur le créneau indiqué sur ADE pour répondre aux questions
        * Une salle est réservée à l'école pour ceux qui ne peuvent pas travailler de chez eux (prévoir ordinateur + écouteurs)
        * Des QCM pour valider la compréhension du cours sont disponibles (bonus jusqu'à 2 points sur le BE)
          + Réaliser l'activité Moodle qui permet de s'inscrire dans le bon groupe de TP pour avoir accès aux QCM sur les cours
          + Faire les QCM sur les cours AVANT les TDs (sinon pas de points bonus)
            ° QCM des cours 1 et 2 AVANT le TD1 et idéalement avant le TP1
            ° QCM du cours 3 AVANT le TD1
            ° QCM du cours 4 AVANT le TD4
      - Pour ceux qui veulent utiliser leur propre machine : installer l'environnement utilisé pendant les TP AVANT le premier TP (voir sujet du TP1 pour les instructions)
      - Attention à ne pas accumuler le retard sur les cours.
